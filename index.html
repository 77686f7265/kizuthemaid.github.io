About

Shinobu was written with simplicity in mind. Even if you are a beginner, the code will make sense to you. A little bit of background in programming is still required, but as long as you know the basics, you can dive in.

Swift & JavaScript

Shinobu is written in both JavaScript & Swift. I'm pretty sure you are wondering why I used two programming languages, well simple: I used node.js to ensure that the bot would be running, while I code the real version in Swift, that way people can still use the bot.

Open Source

Shinobu being open source means that you can run your own bot using the source code! I like open sourcing my projects, so feel free to make your own using the code available here!

Deleting messages is such a pain...But not with Shinobu!

Have tons of messages you want to delete? With shinobu, it's never been easier, just type .purge ### where ### represents the amount of messages you want to delete. :)

Ban/kick people

Shinobu can also ban people! To ban someone simply just type .ban @ThisAnnoyingKid#666. Same thing applies to kick people.
