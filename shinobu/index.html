Shinobu - Discord Bot

A bot for discord, she can interact with people, ban/kick them, and so on. Scroll down to know what she can do.

About

Shinobu was written with simplicity in mind. Even if you are a beginner, the code will make sense to you. A little bit of background in programming is still required, but as long as you know the basics, you can dive in.

Swift

Shinobu is written in Swift, a programming language developed by Apple, and uses Sword, a discord api wrapper(written in Swift, of course). For those who don't know Swift, well let me explain. Swift is a Safe, Fast, and expressive programming language. Swift includes features that make code easier to read and write, while giving the developer the control needed in a true systems programming language.

Open Source

Shinobu being open source means that you can run your own bot using the source code! I like open sourcing my projects, so feel free to make your own using the code available here!

Planned Commands
